# MonteCarloSimulator

Monte is a Python Monte Carlo Simulator. Initally only support for "throwing Pi". More to come.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements.txt
```

## Usage

```python
chmod +x pi.py
```

```python
./pi.py
```
